# 💸 Expense Tracker - Flask & SQLite

A full-stack web application designed to help users track their daily expenses efficiently. This project features a clean, responsive UI and a robust backend to manage financial data.

## 🚀 Features
* **Add Expenses:** Log new expenditures with a description and amount.
* **Data Persistence:** Uses SQLite to store your data locally.
* **Responsive Design:** Styled with Bootstrap for a seamless experience on mobile and desktop.
* **Real-time Updates:** View your transaction history immediately.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Version Control:** Git & GitHub
